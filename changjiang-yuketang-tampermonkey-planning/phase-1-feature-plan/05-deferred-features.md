# Deferred Features

## F16. 自动翻页 / 自动滚动 / 自动点击下一页

**功能目标**：

- 自动推进 PPT 阅读过程

**用户价值**：

- 可能显著减少重复操作

**实现依赖**：

- PPT 完成判定逻辑
- 页面事件机制
- 停留时长规则

**风险等级**：

- 高

**是否涉及自动化**：

- 是，高敏感自动化

**是否需要继续提供页面信息**：

- 是，强依赖

## F17. 自动触发影响学习进度的页面事件

**功能目标**：

- 直接推动平台记录学习进度

**用户价值**：

- 理论上可提高完成效率

**实现依赖**：

- 接口、事件、埋点逻辑

**风险等级**：

- 很高

**是否涉及自动化**：

- 是，高风险自动化

**是否需要继续提供页面信息**：

- 是，且需要更深入数据

## F18. 自动触发打印 / 批量导出 PDF

**功能目标**：

- 大幅降低导出操作成本

**用户价值**：

- 对重度导出场景有价值

**实现依赖**：

- 浏览器打印链路
- 打印预览行为
- 系统级另存为 PDF 行为

**风险等级**：

- 高

**是否涉及自动化**：

- 是

**是否需要继续提供页面信息**：

- 是，强依赖
