# Phase 1 Feature Plan

## 简介

本目录用于单独存放“长江雨课堂油猴插件”一期相关的功能规划文档。

文档内容基于当前已确认的产品边界、优先级和风险约束整理，聚焦“一期功能分层建议”，便于后续继续补充实现依赖、页面信息、测试范围和迭代计划。

## 目录用途

- 汇总一期功能范围
- 按优先级和实施建议分层管理功能
- 为后续实现设计、任务拆解和验收准备提供依据

- `03-should-have-features.md`
  - 一期应该做的功能清单
- `04-optional-features.md`
  - 一期可选做的功能清单
- `05-deferred-features.md`
