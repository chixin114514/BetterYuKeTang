# Optional Features

## F13. 临近截止任务的更丰富提醒规则

**功能目标**：

- 支持按 24 小时、48 小时、3 天等规则标红或分级显示

**用户价值**：

- 更细化的 deadline 管理

**实现依赖**：

- deadline 数据格式稳定

**风险等级**：

- 低

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F14. 本地标记系统

**功能目标**：

- 允许用户额外标记“已看过 / 稍后处理 / 自己已处理”

**用户价值**：

- 补足平台状态不透明的问题

**实现依赖**：

- 稳定的内容唯一标识

**风险等级**：

- 低到中

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F15. PPT 阅读辅助操作面板

**功能目标**：

- 为阅读页增加更直接的操作入口

**用户价值**：

- 提升阅读效率

**实现依赖**：

- 阅读页空间布局
- 翻页控件结构

**风险等级**：

- 中

**是否涉及自动化**：

- 可不涉及，也可扩展到轻度自动化

**是否需要继续提供页面信息**：

- 是
