# Should Have Features

## F9. 课程级任务概览

**功能目标**：

- 在课程页面显示任务总览、未完成统计、PPT/作业/实验分布

**用户价值**：

- 强化“全局感知”

**实现依赖**：

- 内容列表是否完整可获取

**风险等级**：

- 低到中

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F10. 批量打开待处理内容

**功能目标**：

- 批量打开多个未完成内容或某一类内容

**用户价值**：

- 减少逐个打开的重复操作

**实现依赖**：

- 链接结构
- 浏览器弹窗限制
- 平台多标签可用性

**风险等级**：

- 中

**是否涉及自动化**：

- 是，低风险自动化

**是否需要继续提供页面信息**：

- 是

## F11. PPT 阅读状态可视化

**功能目标**：

- 显示当前页、总页数、已读/未读判断、可能的完成状态

**用户价值**：

- 回应“PPT 是否真的完成 100%”的痛点

**实现依赖**：

- PPT 页面结构
- 进度显示区域
- 页码结构

**风险等级**：

- 中到高

**是否涉及自动化**：

- 否，先做可视化

**是否需要继续提供页面信息**：

- 是，强依赖

## F12. 自动跳到未读页

**功能目标**：

- 帮助用户快速定位未读页面

**用户价值**：

- 降低 PPT 阅读过程中的机械操作

**实现依赖**：

- 如何识别未读页
- 是否存在缩略图/目录/页码结构

**风险等级**：

- 中到高

**是否涉及自动化**：

- 是，可控自动化

**是否需要继续提供页面信息**：

- 是，强依赖
