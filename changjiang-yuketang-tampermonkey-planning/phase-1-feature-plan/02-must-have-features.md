# Must Have Features

## F1. 页面识别与上下文识别

**功能目标**：

- 识别当前页面类型、当前课程、当前内容上下文

**用户价值**：

- 这是所有增强能力的基础，确保插件只在正确页面生效

**实现依赖**：

- 页面 URL 结构
- 页面 DOM 特征
- 页面切换方式是否为 SPA

**风险等级**：

- 中

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F2. 课程内容列表增强与内容分类

**功能目标**：

- 对课程内内容进行分类识别，如作业、实验、PPT、其他

**用户价值**：

- 解决“内容太散、不好找”的核心问题

**实现依赖**：

- 内容列表结构
- 内容项中的类型标识、标题、图标、状态字段

**风险等级**：

- 中

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F3. 作业 / 实验聚合看板

**功能目标**：

- 汇总当前课程内待完成作业/实验，展示状态、deadline、优先级

**用户价值**：

- 直接解决当前第一优先级痛点

**实现依赖**：

- 列表页是否有状态字段
- deadline 展示位置
- 作业/实验页面或列表页中的状态来源

**风险等级**：

- 中

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F4. deadline 汇总与临期高亮

**功能目标**：

- 汇总 deadline，并对临近截止项做加红提示

**用户价值**：

- 提升任务管理效率，避免漏交

**实现依赖**：

- deadline 字段是否存在
- 时间格式是否统一
- 时区与状态规则

**风险等级**：

- 低到中

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F5. 快速筛选与快速跳转

**功能目标**：

- 支持按未完成、作业、实验、PPT、临期等维度筛选，并快速跳转

**用户价值**：

- 减少查找和点击成本

**实现依赖**：

- 课程内容列表结构
- 页面跳转方式

**风险等级**：

- 低

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F6. 打印入口识别与 PDF 导出辅助

**功能目标**：

- 帮助用户找到打印入口，提示导出前检查项和打印参数

**用户价值**：

- 直接解决第二优先级痛点

**实现依赖**：

- 打印按钮位置
- 点击后的跳转链路
- 打印页结构
- 是否需懒加载全部页

**风险等级**：

- 中

**是否涉及自动化**：

- 轻度，默认应先做非自动化辅助

**是否需要继续提供页面信息**：

- 是

## F7. 统一增强 UI 容器

**功能目标**：

- 提供统一的增强面板或工具区，承载筛选、汇总、日志入口、快捷操作

**用户价值**：

- 保证功能集中、可发现、可使用

**实现依赖**：

- 页面布局可插入区域
- 是否存在遮挡风险

**风险等级**：

- 低到中

**是否涉及自动化**：

- 否

**是否需要继续提供页面信息**：

- 是

## F8. 配置与日志系统

**功能目标**：

- 提供开关、模式选择、自动化日志、执行记录

**用户价值**：

- 满足“可控、可见、可追踪”的已确认要求

**实现依赖**：

- 功能分级规则
- UI 放置方式
- 本地存储方案

**风险等级**：

- 低

**是否涉及自动化**：

- 间接支持自动化

**是否需要继续提供页面信息**：

- 部分需要
